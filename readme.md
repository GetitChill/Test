# Overview
Text based game. There's players and monsters. The number of monsters is random, and the number of players can be set. The monsters move around the screen, and the aim of the player(s) is to avoid being touched.

# Learning Objectives
This is a **simple** project with the primary goal to learn effective software development, and c++.
    - CLI git
	- Linux development using an editor.
	- CMAKE and Catch2.
	- Strings (IOstream and c-style), File IO, Pointers
    - Random number generation (merssene)
